# Гиря

Программа тренировок с GIF-техникой. Работает офлайн в одном HTML-файле.

**Сайт:** https://mat3eO421.github.io/gyrya/

## На экран iPhone

1. Открой ссылку в **Safari**
2. Нажми **Поделиться** → **На экран «Домой»**
3. Имя: **Гиря** → **Добавить**

## Обновить GIF локально

1. Правь `programs.json` (поле `"gif"`)
2. Запусти `python3 build.py`
3. Закоммить и запушь обновлённый `index.html`
